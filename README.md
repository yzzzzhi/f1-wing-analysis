# F1 Wing Analysis

A project exploring Formula 1 rear wing aerodynamics and performance trends using Python data analysis and visualisation.

## Goals
- Analyse wing geometry trends
- Compare aerodynamic concepts
- Visualise performance differences

## Tools
- Python
- Pandas
- Matplotlib
- Aerosandbox

## Engineering Conclusions

This project investigated the aerodynamic behaviour of several NACA airfoils under inverted-wing operating conditions representative of motorsport aerodynamic devices.

Results showed that the symmetric NACA0012 airfoil achieved both the highest downforce coefficient and the best downforce-to-drag ratio among the airfoils studied. In contrast, increasingly cambered airfoils demonstrated significantly poorer aerodynamic efficiency under negative angles of attack.

The drag polar analysis further demonstrated the nonlinear trade-off between downforce generation and drag accumulation, highlighting the importance of balancing cornering performance against aerodynamic resistance in motorsport wing design.

These findings suggest that symmetric airfoils may provide more favourable flow behaviour and aerodynamic efficiency for inverted aerodynamic loading conditions.
